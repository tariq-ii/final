Original Approved Plan

The training program was structured into three primary phases:

  1. IT Support Operations (Weeks 1-4)

    • Call center operations
    • Technical support systems
    • Incident analysis
  2. Web Development Fundamentals (Weeks 5-7)
    • Front-end development basics
    • JavaScript and API integration
    • Angular framework development
  3. Project Implementation (Weeks 8-9)
    • Application development
    • System integration
    • Final project completion

Implemented Schedule

The actual implementation closely followed the original plan, with adjustments made to accommodate real-world project needs and learning progression.

Developed a comprehensive weather dashboard web application using Angular framework and JavaScript, integrated with weather API services.

Project Features
